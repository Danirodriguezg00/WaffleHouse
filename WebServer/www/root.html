You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /github?query=users/"Username"/repos -- returns information on Username's repos
  • /multiply?num1=X&num2=Y -- replace X and Y with two integers to return their product
  • /fun?width=5&height=5 -- returns shape made of *'s with specified demensions
  • /story?name=daniel&adj1=slimy&adj2=slippery&verb1=died&verb2=boogied -- Returns a funny story with inputted parameters

How to USE

  • FUN: fun?width="in"&height="int"
  • STORY: story?name="name"&adj1="adj"&adj2="adj"&verb1="PastTenseVerb"&verb2="PastTenseVerb"
